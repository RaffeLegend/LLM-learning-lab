# CLAUDE.md

## 项目概述

这是一个**多模态大语言模型（Multimodal LLM）**的个人学习笔记仓库。目标是系统性地记录学习过程，包括论文阅读、代码实验、概念梳理和项目实践。

## 对 Claude 的指引

### 你应该做的

- **用中文**撰写所有笔记和说明
- 解释概念时兼顾**直觉理解**和**技术细节**，先直觉后公式
- 论文笔记中主动对比相关工作，帮助构建知识图谱
- 在笔记末尾添加"个人理解与思考"部分，鼓励批判性思考
- 代码实验附带清晰注释，解释每一步的目的
- 主动建议将新内容与已有笔记进行关联
- 使用 Mermaid 图表辅助解释架构和流程

### 你不应该做的

- 不要跳过数学推导，但要用通俗语言解释每一步的含义
- 不要只罗列信息，要有结构化的归纳和总结
- 不要忽略实验细节（超参数、训练技巧等）
- 不要在笔记中使用未经解释的缩写

### 代码风格

- Python 代码遵循 PEP 8
- 实验代码使用 PyTorch
- Notebook 中每个 cell 有简短说明
- 关键实验记录超参数和结果
